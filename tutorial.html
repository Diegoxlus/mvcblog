Implementación de un blog en PHP siguiendo el patrón Modelo-Vista-Controlador (MVC) basado en acciones, Orientación a Objetos y Mapeadores Objeto-Relacional

Daniel Glez-Peña

Introducción

  • El ejemplo consiste en un Blog con comentarios, es decir, posts y comentarios.
  • Vamos a emplear las capacidades de Orientación a Objetos de PHP 5.
  • Vamos a user MySQL, pero no directamente a través de MySQLi a través de la API PDO (PHP Data Objects)

El Modelo

Para la implementación del modelo vamos a usar objetos de dominio (p. ej: Post) junto con clases auxiliares para mapear estos objetos (clases Mapeador) con la base de datos.
